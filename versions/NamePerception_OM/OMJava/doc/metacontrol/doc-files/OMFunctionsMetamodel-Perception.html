Perception Models:


FunctionGrounding

possible status:
  • OK
  • ERROR - internal error
  • FAILURE - external error
  • PERMANENT_FAILURE - UNACHIEVABLE required objective (external dependency)
perception model:
    (internal)
     - if any binding with the realiser (GoalAtomTracker) in error -> ERROR
    (integration with external)
     - if any required objective is UNACHIEVABLE -> PERMANENT_FAILURE
     - if not ERROR && not PERMANENT_FAILURE
          if any required objective in ERROR -> FAILURE
          else -> OK



Objective

possible status:
  • ADDRESSED - there is an associated fg in OK status
  • ACHIEVED - this is the ok status - not used!!!
  • ERROR - current fg has not achieved it
  • UNACHIEVABLE - its function status is UNAVAILABLE
perception model:
     - if the objective has no realiser or it is in ERROR -> ERROR
     - if its type (a function) is UNAVAILABLE and current realiser is in ERROR -> UNACHIEVABLE
     - else -> ADDRESSED


Function

possible status:
  • AVAILABLE - here is at least one of its  FSpec REALISABLE
  • UNAVAILABLE - no FSpec is REALISABLE
perceptual model:
     - if any realisation (fspec) REALISABLE -> AVAILABLE
     - if none -> UNAVAILABLE


FunctionSpecification

posible status:
  • REALISABLE
  • UNREALISABLE

perceptual model:
     - (internal) if any of the role types is unavailable -> UNREALISABLE
     - (external) if any of the required functions is UNAVAILABLE -> UNREALISABLE
